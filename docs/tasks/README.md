# Completed Tasks Archive

This directory (`/docs/tasks/`) serves as an archive for completed development tasks. 

Whenever a feature or bug fix is finished, its `task.md`, `walkthrough.md`, and `verification_report.md` (renamed to `verification.md`) should be moved here into a subfolder named after the task or branch (e.g., `/docs/tasks/wfp-21-add-login/`).

**Archival Purpose:**
- Preserves context for future developers.
- Prevents root directory clutter.
- Maintains a verifiable history of implementation choices.
